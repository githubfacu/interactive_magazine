
# Art & Music Interactive Magazine

Simple interactive digital magazine focused on visual experience, DOM manipulation, and client-side state management.

## Overview
This project simulates a digital magazine where users can explore music artists, watch video performances, and interact through a voting system.  
After login, the user accesses a gallery-style interface displaying artist profiles, embedded videos, and a live ranking based on user interactions.

## Features
- Responsive grid gallery background  
- Layout built with CSS Grid and Flexbox  
- Lightweight project without frameworks  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript  

## Purpose
Portfolio project aimed at demonstrating front-end layout skills, responsive design, and visual composition.
